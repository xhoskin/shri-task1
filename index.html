Сверстайте всплывающее окно (попап) — элемент, который показывается поверх содержимого веб-страницы. Он должен всегда находиться в её центре, при этом размер контента может меняться. Предложите максимальное количество решений и объясните, какие у каждого из них есть недостатки и преимущества. Результат пришлите в виде ссылки на GitHub.

  1. С помощью margin

    Отступ слева задается автоматически (margin:auto)

    Отступ сверху считается js-ом (margin-top)

    Преимущества

    1. минимум расчетов на js, не нужно рассчитывать отступ по горизонтали;
    2. если изменится ширина попапа, то НЕ нужно пересчитывать отступ по горизонтали чтобы попап оставался по центру экрана;
    3. легко делать адаптивную шируну попапа с помощью @media-запросов

    Недостатки

    1. нельзя прикрутить перетаскивание попапа
    2. нужно пересчитывать margin-top, если изменятся высота окна браузера или самого попапа

  2. С помощью left и top

    left и top расчитывются скриптом, ( <размеры_окна> - <размеры_попапа> ) / 2

    Преимущества

    1. можно прикрутить перетаскивание попапа

    Недостатки

    1. нужно пересчитывать top и left если изменятся размеры окна браузера или самого попапа

  3. С помощью translate

    top и left - 50% , transform:translate(-50%, -50%)

    Преимущества

    1. не требует рассчетов, можно полностью описать в css
    2. можно прикрутить перетаскивание попапа, хотя это реализовывается с трудом

    Недостатки

    1. не будет работать в ie8 и менее
    2. требует вендорных префиксов

  4. С помощью display:table

    Преимущества

    1. не требует рассчетов, можно полностью описать в css

    Недостатки

    1. требует дополнительные обертки вокруг попапа
    2. нельзя сделать перетаскивание попапа
    3. не будет работать в ie7

  5. С помощью position:fixed и фиксированных размеров слоя

    Преимущества

    1. не требует рассчетов, можно полностью описать в css

    Недостатки

    1. обязательно нужно задавать ширину и высоту попапа
    2. нельзя автоматически подобрать высоту попапа по содержимому, обязательно нужно ее проставить, значит внизу попапа может образоваться ненужная пустота
    3. если переполнится содержимое попапа, то или появится прокрутка, или нужно изменять высоту попапа
